# AKAGAMI v2.0.0-beta — Command Deck Release

## Highlights

- Command Deck CLI UI
- dry-run recon planning
- strict target validation
- offline/no-LLM mode
- tool diagnostics
- endpoint inventory
- report generation with brand assets
- operator approval and stage-gate workflow
- 436 passing tests

## Safety Notes

- authorized targets only
- active scans require explicit approval/scope
- no-LLM mode supported for deterministic/offline operation

## Quick Start

```bash
python -m venv venv
source venv/bin/activate
pip install -e .
akagami tools doctor
akagami recon -t example.com -C Example --passive-only --dry-run
AKAGAMI_NO_LLM=1 akagami recon -t example.com -C Example --passive-only
```
